These are the components used to build the stations/cabot and stations/riverway
models.

- `riverway_station.sdf`
  - A set of three wheeled carts welded together with an affixed camera pole.

- `station_arm_cart_gantry.sdf`
- `station_gantry_base_plate.sdf`
  - A gantry for a Panda arm that affixes to the wheeled carts and a plate that
    serves as the interface between gantry and Panda arm.

- `station_cherry_red_table_top_compliant.sdf`
- `station_dark_brown-table_top_compliant.sdf`
- `station_natural_brown_table_top_compliant.sdf`
  - The manipulation table (with various finishes).
